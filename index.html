Frequently Asked Questions

My code is broken! Can you fix it?

If it is a small mishap or code breaking that is not of your doing (image or music hosting problems), Kingdom Corp. will gladly fix the issue for you for free. But if it changes a large part of your code (even if it breaks because of RPC updates) and will take a while to fix, there will be a fee equal to a Fundamental Code Edits Add On.

I want a code edit but you didn't make the original profile. Can I still avail of that add on?

Yes, but under the condition that you and Kingdom Corp. both ask permission individually from the original coder to alter the profile. If they refuse, you may not avail of the add on.

Can I request a profile instead of commission a profile?

We will announce when we are open for requests and how many slots we are going to take. However, it is no guarantee that we will accept your request, but we will evaluate which profiles appeal to us the most.

Do you provide code tutorials or code appraisals?

We may answer a question or two, or give passing advice on profile improvements but anything in depth is not something we can do with our current schedule. If one da we do offer this service, it will show up in our Add Ons section.
